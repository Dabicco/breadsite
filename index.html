Welcome to the Bread Gaming Server!

We love cats! We also love Minecraft and host a lobby server that offers various minigames, including:

  • Wither Storm Anarchy Survival
  • Explorers SMP (custom terrain world)
  • Lemon SMP (private world)
  • An event portal for all kinds of events across many Discord servers, such as EEGAR's server and The Decayed Team.

In addition to Minecraft, we embrace a game called Bopl Battles, a fun couch/online multiplayer game where slimes run around and battle. We occasionally host tournaments and other events for this game.
