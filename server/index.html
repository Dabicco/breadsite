The Official DDMM Server

Via our Discord Server, we regularly* host events, ranging from a nice happy SMP to a full on out CWSM + Create Mod Modpack with unique challenges and adventures! Our server's hardware has been tested up to 100 players, and has worked flawlessly anyway. The server software includes a forked version of the project PaperMC made by Dabicco and a Forked version of Velocity (also made by PaperMC) named Embeddium, also made by Dabicco. These combined factors generate the most lag-free and seamless Minecraft server experience we can get.

Bedrock Edition

Follow these steps to join:

  • Open Minecraft and navigate to Play Menu → Multiplayer → Add Server.
  • Enter the following information:
    • IP: play.breadgames.online
    • Port: 19132 (default port)

Java Edition

Follow these steps to join:

  • Navigate to Multiplayer → Add Server/Direct Connect.
  • Enter the following information:
    • IP: play.breadgames.online
  • Enable Server Resource Packs!
