Bedrock Edition

Follow these steps to join:

  • Open Minecraft and navigate to Play Menu → Multiplayer → Add Server (you may need to scroll down in the server list).
  • Enter the following information:
    • IP: play.breadgames.online
    • Port: 19132 (default port)

Java Edition

Follow these steps to join:

  • Navigate to Multiplayer → Add Server/Direct Connect.
  • Enter the following information:
    • IP: server.breadgames.online
  • Enable Server Resource Packs.
