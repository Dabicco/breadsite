Dabiccan Language Rulebook

Introduction

Dabiccan is a constructed language based on English with deterministic letter swaps, vowel manipulations, and Italian-style endings. It is designed to be decodable by anyone who knows English and the fundamental rules described here.

1. Word Order

  • Keep English word order: Subject → Verb → Object.
  • Adjectives follow nouns.
  • Optional Italian replacements:
    • the → il / la
    • and → e
    • my → mio / mia

2. Consonant Swap

  • Swap consonants in pairs, left-to-right.
  • If there is an odd number, the last consonant stays in place.
  • Only alphabetic consonants are swapped.
  • Example: cat → tcao, dog → gdoa

3. Vowel Swap

  • Swap vowels in pairs, left-to-right.
  • If there is an odd number, the last vowel stays in place.
  • Example: friend → rfiedni, bread → baredo

4. Italian-style Endings

  • Forced ending based on the first letter of the word:
    • a–f → -a
    • g–l → -e
    • m–r → -i
    • s–z → -o
    • Single-letter words → -o
  • Capital letters always appear at the first letter of the Dabiccan word.
  • Punctuation (.,!?) inside words is moved to the end of the word.

5. Encoding Steps

  1. Take the English word.
  2. Swap consonants in pairs.
  3. Swap vowels in pairs.
  4. Add the forced Italian ending.
  5. Adjust capitalization and punctuation.

6. Decoding Steps

  1. Remove the Italian ending (last vowel added).
  2. Reverse vowel swaps in pairs.
  3. Reverse consonant swaps in pairs.
  4. Move the capital letter back to its original position if needed.
  5. Reposition punctuation if it was moved.

7. Examples

  • English → Dabiccan:
    • hi bro → Hia Rboa
    • The cat and dog eat bread → Il Tcao e Gdoa eato Baredo
    • I hope you enjoy yourself → I Hpoe You Eonjy Yrouselfi

8. Notes

  • Dabiccan is deterministic: the same English word always produces the same Dabiccan word.
  • It is designed to be reversible, making translation back to English possible.
  • Capitalization and punctuation adjustments improve readability and maintain grammatical structure.
