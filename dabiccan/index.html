Dabiccan Rulebook & Translator

1. Word Order

Keep English word order (Subject → Verb → Object). Adjectives come after nouns.

Optional Italian-style replacements:

  • the → il / la
  • and → e
  • my → mio / mia

2. Consonant Swap

Swap consonants in pairs, left to right. Odd number → last consonant stays.

Example: cat → tca, dog → gdo

3. Vowel Swap

Swap vowels in pairs, left to right. Odd number → last vowel stays.

Example: friend → freidn, bread → bared

4. Italian-style Ending (Forced)

Endings are based on first letter of the word:

  • a–f → -a
  • g–l → -e
  • m–r → -i
  • s–z → -o
  • Single-letter words → -o

5. Step-by-Step Encoding

  1. Take the English word.
  2. Swap consonants in pairs.
  3. Swap vowels in pairs.
  4. Add forced Italian ending based on first letter.
  5. Keep word order English.
  6. Replace articles/prepositions optionally with Italian words.

6. Examples

  • cat → tcao
  • dog → gdoa
  • bread → baredo
  • friend → rfiedni
  • The cat and dog eat bread → Il tcao e gdoa eato baredo
  • I love my friend → I lovo mio rfiedni
